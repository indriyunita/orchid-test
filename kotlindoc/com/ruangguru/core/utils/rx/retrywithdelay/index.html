RetryWithDelay

class RetryWithDelay

RetryWithDelay use with Rx2 operator Observable.retryWhen and it will do re-subscribe the stream if error occur but it will retry IF the condition is fulfilled. But retry process will stop IF the error is UnknownHostException which indicate there is no internet
